# Gitのよく使うコマンド

このファイルは、Gitの日常的な操作で私的に頻繁使用するコマンドをまとめたものです。

## ブランチ操作

### `git branch`
ローカルブランチの一覧を表示
```
git branch
```

### `git branch -D`
ローカルブランチを強制的に削除
```
git branch -D <branch_name>
```

## 日常的なワークフロー

### `git status`
ワーキングディレクトリとステージングエリアの状態を表示
```
git status
```

### `git add`
ファイルへの変更を次のコミットのためにステージングエリアに追加
```
# すべての変更をステージング
git add .

# 特定のファイルをステージング
git add <file_name>
```

### `git commit`
ステージングされた変更をリポジトリの履歴に記録
メッセージ付きでコミット
```
git commit -m "ここにコミットメッセージを記述"
```

### `git diff`
ワーキングディレクトリとステージングエリアの差分、またはコミット間の差分を表示
```
# ワーキングディレクトリとステージングエリアの差分
git diff

# ステージングエリアと最新コミットの差分
git diff --staged
```
